# Flash Coin Flipper

A simple web application built with Python and Flask that simulates a coin flip.

## How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/ramizhk/flask-coinflip.git](https://github.com/ramizhk/flask-coinflip.git)
   cd flask-coinflip
   
2. **Install all dependencies:**
   ```bash
   python -m venv venv
   
3. **Run the app**
   ```bash
   python app.py

4. Open ```http:/127.0.0.1:5000/``` in your browser
   
# On Windows:
```.\venv\Scripts\Activate.ps1```

# On macOS/Linux:
```source venv/bin/activate```
